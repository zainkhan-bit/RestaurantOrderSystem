# Restaurant Order System 🍽️

A complete relational database project for a Restaurant Order System.
Built as a final project for Database Management Systems course.

## 📋 Project Overview
This database manages all restaurant operations including:
- Menu categories and items
- Customer records
- Employee management
- Table assignments
- Order processing
- Payment handling

## 🗄️ Database Tables
| Table | Description |
|-------|-------------|
| MenuCategory | Menu categories (Starters, Main Course etc.) |
| MenuItem | All menu items with prices |
| Customer | Customer information |
| Employee | Staff records (Waiter, Chef, Manager, Cashier) |
| RestaurantTable | Physical restaurant tables |
| Orders | Customer orders |
| OrderItem | Individual items within each order |
| Payment | Payment records |

## 📁 Files
- `restaurant_order_system.sql` - Complete SQL file with schema, data, views and queries

## 🔧 How to Run
1. Open MySQL Workbench
2. Open `restaurant_order_system.sql`
3. Press Ctrl + Shift + Enter to run
